## 💫 About Me

I'm a versatile developer and hands-on tinkerer who loves bridging the gap between clean, logical software and physical hardware. From training neural networks and writing clean web code to wiring up physical microcontrollers and writing embedded loops, I thrive on building things that work seamlessly.

* 🛠️ **Current Focus:** Building responsive web applications, optimizing embedded systems, and diving deeper into data science.
* 🤖 **IoT & Hardware:** Love working with physical circuits, sensor arrays, and microcontroller logic.
* ✍️ **Philosophy:** Keep code modular, commit messages semantic, and hardware connections grounded.
